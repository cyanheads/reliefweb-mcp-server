# reliefweb-mcp-server

Humanitarian crisis data, disaster reports, and situation updates via the OCHA ReliefWeb API.

## Data source

- **ReliefWeb API v1** — humanitarian reports, disaster data, country profiles, jobs, training
- **Auth**: None required
- **Rate limits**: Generous, public API

## Why it earns its keep

The canonical source for humanitarian information worldwide. Disaster reports, situation updates, assessments, maps — used by aid workers, journalists, researchers, and policy makers. Niche but genuine public good. No other MCP server covers this domain.

## Target users

- Humanitarian aid workers and organizations
- Journalists covering crises and disasters
- Researchers studying conflict, displacement, disaster response
- Policy makers and development professionals
- Agents combining with earthquake/NWS for disaster context

## Scope

- Read-only
- Report search (situation reports, assessments, analyses, maps)
- Disaster search and metadata
- Country/territory profiles and status
- Job listings in humanitarian sector
- Training/learning opportunities
- Source organization browsing
- Filter by disaster type, country, theme, date range
