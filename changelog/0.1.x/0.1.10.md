# 0.1.10 — 2026-06-15

## Added

- **`instructions`** on `createApp()` — session-level orientation sent on every `initialize`: tool-family overview, required `RELIEFWEB_APP_NAME`, identifier shapes (ISO3 country codes, numeric report/disaster IDs, organization shortnames), the search-then-fetch workflow, and the 1,000 calls/day quota.

## Changed

- **Plugin manifest display identity** — `.claude-plugin/plugin.json` (`name`, `mcpServers` key), `.codex-plugin/plugin.json` (`interface.displayName`), and `.codex-plugin/mcp.json` (server key) now use the unscoped `reliefweb-mcp-server` so the agent-facing display name matches the hyphenated repo name. Install `args` stay `@cyanheads/reliefweb-mcp-server` (the published npm package).

### Dependencies

- `vitest` ^4.1.8 → ^4.1.9
